# Cat Feeding App

This is a Java GUI application that allows users to:
- Add cats
- Schedule feedings
- Track upcoming feedings (Queue)
- Track feeding history (Linked List)

## How to Run

1. Open the project in an IDE (Eclipse)
2. Make sure all files in the `src` folder are included
3. Run the main class:

driver.MainGUI

## Notes
- The program uses:
  - Linked List (custom implementation)
  - Queue (Java LinkedList)
  - Sorting (Insertion Sort logic)

- If using DatePicker:
  Make sure the `lib/LGoodDatePicker.jar` is added to the project build path
