# Cat Feeding App

This is a Java GUI application that allows users to:
- Add cats
- Schedule feedings
- Track upcoming feedings (Queue)
- Track feeding history (Linked List)

## How to Run

- Open the project in your IDE. (Preferably Eclipse)
- Ensure the src folder is configured as the source root.
- Add lib/LGoodDatePicker.jar to the project build path.
- Run the main class: driver.MainGUI

## Notes
- The program uses:
  - Linked List (custom implementation)
  - Queue (Java LinkedList)
  - Sorting (Insertion Sort logic)

- If using DatePicker:
  Make sure the `lib/LGoodDatePicker.jar` is added to the project build path
